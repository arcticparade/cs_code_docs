# CS Code Router Docs

Public documentation for CS Code Router.

This repository contains the customer-facing docs site for:

- using the web app
- authenticating with the platform
- calling the public API
- using the CLI
- understanding routing and security behavior

## Read the docs

Start here:

- `docs/index.md`
- `docs/quickstart.md`
- `docs/user-guide.md`
- `docs/api-reference.md`

## What this repository is for

This is the public docs repository only.

It does not contain the private application codebase or internal deployment runbooks.

## Publishing

The docs site is built with VitePress and published through GitHub Actions and GitHub Pages.
