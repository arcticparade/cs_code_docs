What these docs cover ​

This site is for people using the product, integrating with the API, or automating from the CLI.

You will find:

  • quick onboarding guidance
  • public API and CLI references
  • workspace behavior and routing explanations
  • authentication and security guidance for normal product use

Start here ​

  • quickstart.md for the fastest product onboarding path
  • user-guide.md for the core product loop and workspace workflows
  • workspace-reference.md for the Studio, Teams, Billing, policies, and settings surfaces

Core references ​

  • api-reference.md for the API map and endpoint-family navigation
  • cli-reference.md for install, auth, routing, and execution from the terminal
  • auth-reference.md for session auth, API keys, and SSO behavior
  • model-routing.md for scoring, fallback behavior, and optimization logging
  • security-features.md for prompt protection, scoped access, and auditability
